# 雷达图样式修改计划

## 1. 问题分析

雷达图没有正确显示分数排布，表现异常。从用户提供的图片可以看到，目标雷达图是一个菱形网格样式，有清晰的刻度和数据填充。

## 2. 可能的原因

1. **canvas尺寸问题**：实际渲染尺寸与设置不符
2. **网格样式问题**：当前配置生成的是圆形网格，而不是菱形网格
3. **数据显示问题**：数据可能没有正确绘制
4. **颜色和填充样式问题**：当前样式不符合目标效果

## 3. 修复计划

### 3.1 调整canvas尺寸
- 确保canvas元素有足够的尺寸
- 修复CSS样式，确保canvas能正确填充容器

### 3.2 修改雷达图配置
- 将圆形网格改为菱形网格
- 调整刻度和标签样式
- 优化颜色和填充效果
- 确保数据正确显示

### 3.3 优化样式
- 参考目标图片调整颜色方案
- 优化填充效果
- 调整网格线样式

## 4. 实现步骤

1. 修改drawRadarChart函数，调整雷达图配置
2. 优化CSS样式，确保canvas尺寸正确
3. 测试雷达图的渲染效果
4. 确保PDF导出时雷达图能正确显示

## 5. 预期效果

- 雷达图显示为菱形网格
- 有清晰的刻度值（20, 40, 60等）
- 数据区域有明显的填充效果
- 颜色方案符合目标样式
- 数据正确显示在雷达图上
