# 移除测评页面滚动条计划

## 问题分析
当前测评页面的`.questions-list`类设置了`max-height`和`overflow-y: auto`，导致题目区域出现独立滚动条。用户要求取消单独滚动，确保所有内容在同一视窗内完整显示。

## 解决方案
1. 修改CSS样式，移除`.questions-list`的滚动条
2. 调整布局，确保所有题目和按钮在同一视窗内
3. 优化题目和选项的间距，确保10道题能完整显示

## 具体修改

### 1. 修改`.questions-list`样式
- 移除`max-height`约束
- 移除`overflow-y: auto`属性
- 调整内边距和外边距

### 2. 调整`.step-page`布局
- 确保flex布局能正确容纳所有内容
- 调整最小高度，确保内容不溢出

### 3. 优化题目样式
- 调整题目和选项的间距
- 确保选项文本清晰可读

## 代码修改点

### CSS样式修改
```css
/* 修改前 */
.questions-list {
    max-height: calc(100vh - 240px);
    overflow-y: auto;
    padding: 0 20px;
    margin-bottom: 20px;
}

/* 修改后 */
.questions-list {
    padding: 0 20px;
    margin-bottom: 20px;
}
```

### 验证
- 确保所有10道题在同一视窗内完整显示
- 无独立滚动条
- 底部按钮与题目区域保持整体性
- 页面布局美观，内容清晰可读

## 预期效果
用户在测评页面可以看到所有10道题和底部按钮，无需单独滚动题目区域，所有内容在同一视窗内完整显示。
